## 🔍 数据说明

- **数据来源**：东方财富网公告（通过AkShare抓取）+ 巨潮资讯网PDF
- **收录范围**：公告标题含"补税/补缴税款/缴纳税款"关键词
- **金额来源**：2026年数据来自公告原文+新闻报道；2025年金额通过PDF文本提取+人工修正
- **更新频率**：手动更新，随新公告发布持续补充
- **注意事项**：部分金额为PDF提取结果，可能存在误差，建议以公告原文为准

## 📈 核心发现

| 指标 | 2025年（全年） | 2026年（1-6月） |
|------|:----------:|:------------:|
| 公告公司数 | 73家 | 87家 |
| 补税总额 | ~20亿元 | ~88亿元 |
| 亿元级公司 | 3家 | 18家 |
| 最大单笔 | 山东黄金 7.38亿 | 北大荒 14.1亿 |

2026年仅半年时间，补税公司数和金额均已大幅超过2025年全年水平，主因是**金税四期全面落地**和**《增值税法》正式实施**带来的监管升级。

## 🛠 技术栈

- 数据抓取：Python + AkShare
- 可视化：Chart.js (纯前端，单HTML文件)
- 托管：GitHub Pages
- PDF提取：PyPDF2
